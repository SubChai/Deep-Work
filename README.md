### 📥 **راهنمای نصب (گام‌به‌گام)**

#### **مرحله 1: باز کردن Apps Script**
1. فایل Google Sheet خود را باز کنید
2. **Extensions** > **Apps Script**
3. کد پیش‌فرض را پاک کنید

#### **مرحله 2: کپی کردن کد**
1. تمام کد از Artifact بالا را کپی کنید
2. در ویرایشگر Paste کنید
3. **Save** کنید (Ctrl+S)

#### **مرحله 3: تنظیم تریگر**
1. آیکون **⏰ Triggers** (سمت چپ)
2. **+ Add Trigger**
3. تنظیمات:
   - Function: `onEdit`
   - Deployment: `Head`
   - Event source: `From spreadsheet`
   - Event type: `On edit`
4. **Save** و مجوزها را بدهید

#### **مرحله 4: بارگذاری مجدد شیت**
1. به شیت برگردید
2. صفحه را **Refresh** کنید (F5)
3. منوی **🎯 Activity Tracker Pro** باید ظاهر شود

### 🎨 **راهنمای رنگ‌بندی**

| رنگ | معنی | محدوده ساعت |
|-----|------|-------------|
| 🟢 سبز روشن | کار کم | < 4 ساعت |
| 🟡 زرد | کار متوسط | 4-8 ساعت |
| 🟠 نارنجی | کار زیاد | 8-12 ساعت |
| 🔴 قرمز | کار خیلی زیاد | > 12 ساعت |

**نکته:** برای اعمال رنگ‌بندی به همه ردیف‌ها: **منو > 🎨 رنگ‌بندی خودکار**

### 📈 **گزارش‌گیری حرفه‌ای**

#### **1. گزارش روزانه:**
```
منو > 📈 گزارش‌ها > 📅 گزارش روزانه
```
- تاریخ شمسی را وارد کنید (مثلاً: `1404/08/02`)
- لیست تمام فعالیت‌های آن روز
- مجموع ساعات کاری روز

#### **2. گزارش هفتگی:**
```
منو > 📈 گزارش‌ها > 📆 گزارش هفتگی
```
(این قابلیت در نسخه‌های بعدی اضافه می‌شود)

#### **3. Export به PDF:**
```
منو > 📈 گزارش‌ها > 📑 Export به PDF
```
یا از منوی Google Sheets:
```
File > Download > PDF Document
```

### 💾 **مدیریت Backup**

#### **ایجاد Backup:**
```
منو > 💾 پشتیبان‌گیری > 💾 Backup الان
```
- خودکار یک شیت جدید با نام `Backup_تاریخ_زمان` ایجاد می‌شود

#### **بازیابی:**
```
منو > 💾 پشتیبان‌گیری > 📥 بازیابی Backup
```
- شیت Backup مورد نظر را پیدا کنید
- محتوای آن را به شیت اصلی کپی کنید

#### **حذف Backup‌های قدیمی:**
```
منو > 💾 پشتیبان‌گیری > 🗑️ حذف Backup‌های قدیمی
```
- Backup‌های قدیمی‌تر از 30 روز حذف می‌شوند

### 🔍 **جستجوی پیشرفته**

```
منو > 🔍 ابزارها > 🔎 جستجوی پیشرفته
```

**می‌توانید جستجو کنید:**
- بر اساس موضوع فعالیت
- بر اساس تاریخ شمسی
- هر عبارتی در فعالیت‌ها

**مثال:**
```
جستجو: "پایتون"
نتیجه: تمام فعالیت‌های مرتبط با پایتون
```

### 🎯 **تنظیم اهداف**

```
منو > 🔍 ابزارها > 🎯 تنظیم اهداف
```

**نحوه کار:**
1. هدف ساعت کاری روزانه را وارد کنید (مثلاً: `8`)
2. سیستم هشدار می‌دهد اگر:
   - بیش از حد کار کنید
   - کمتر از هدف کار کنید

---

### 🔔 **هشدارهای خودکار**

**هشدار کار زیاد:**
```
⚠️ هشدار: 12.5 ساعت کار در یک روز!
```
- وقتی بیش از 12 ساعت در روز کار کنید

**هشدار کار کم:**
```
ℹ️ توجه: فقط 1.5 ساعت کار ثبت شده
```
- وقتی کمتر از 2 ساعت در روز کار کنید

```javascript
ALERTS: {

}
```

### 🐛 **عیب‌یابی**

#### **مشکل: منو ظاهر نمی‌شود**
**راه حل:**
1. شیت را Refresh کنید (F5)
2. از Apps Script دوباره Save کنید
3. Logout و Login کنید

#### **مشکل: تاریخ وارد نمی‌شود**
**راه حل:**
1. تریگر onEdit را چک کنید
2. مطمئن شوید در ستون صحیح تایپ می‌کنید
3. از Execution log خطاها را بررسی کنید

#### **مشکل: رنگ‌بندی کار نمی‌کند**
**راه حل:**
1. از منو: `🎨 رنگ‌بندی خودکار`
2. یا `🔄 بازمحاسبه همه`

```
- مجموع امروز: 6:30 ساعت
- رنگ زرد (کار متوسط)
```
